# 🌸 ИИ-ассистент для салона красоты «Нежно к себе»

> **Проект автоматизации клиентского взаимодействия в салоне красоты с помощью ИИ-ассистента на базе платформы Савви и модели Qwen.**

---

## 📌 Описание проекта

ИИ-ассистент для салона красоты **«Нежно к себе»** — это интеллектуальный чат-бот, созданный для повышения качества клиентского сервиса и автоматизации рутинных задач: от консультирования до записи на процедуры.  
Бот умеет вести естественный диалог, отвечать на вопросы по услугам, ценам, акциям и специалистам, а также оформлять запись клиентов с автоматическим добавлением встречи в **Google Календарь**.

Проект реализован с использованием:
- **Qwen** — для подготовки структурированной базы знаний и системного промпта
- **[Савви](https://suvvy.ai/)** — платформы для создания и развертывания ИИ-ассистентов с поддержкой интеграций и сложных сценариев

---

## 🚀 Ключевые функции

| Функция | Описание |
|--------|----------|
| 🧠 **Осмысленное ведение диалога** | Бот поддерживает контекстный, естественный диалог на основе LLM (large language model), адаптируясь к стилю общения клиента. |
| 📚 **Ответы по базе знаний** | Отвечает на типовые и специфические вопросы: виды процедур, длительность, противопоказания, состав продуктов, отзывы, акции и т.д. |
| 💇 **Консультирование по услугам** | Подбирает услуги по запросу клиента («что подойдёт для сухой кожи?», «есть ли ламинирование ресниц?») с учётом актуальной информации салона. |
| 📅 **Запись на услугу** | После согласования деталей бот собирает: имя клиента, контакт (телефон/email), желаемое время, услугу и мастера — и создаёт бронирование. |
| 🗓️ **Интеграция с Google Календарём** | Запись автоматически добавляется в календарь администратора/мастера — с уведомлением клиенту. |

---

## 🛠 Используемые технологии

| Компонент | Назначение |
|----------|------------|
| **[Qwen](https://qwenlm.github.io/)** | Генеративная языковая модель Alibaba. Использовалась для: <br> • Составления структурированного системного промпта <br> • Обработки и превращения текстов салона (прайсов, описаний услуг, FAQ) в оптимизированную базу знаний |
| **[Савви](https://suvvy.ai/)** | Платформа для создания ИИ-ботов. Обеспечивает: <br> • Инференс на надёжных LLM <br> • 100% покрытие запросов (никаких «не знаю») <br> • Точность ответов ≥95% <br> • Интеграции с внешними API (Google Calendar, CRM, мессенджеры) |
| **Google Calendar API** | Для синхронизации записей и предотвращения пересечений времени. |

## ✅ Преимущества решения

- ⏱ **Экономия времени администратора** — до 70% входящих запросов обрабатывается автоматически  
- 📈 **Повышение конверсии** — мгновенный ответ 24/7, удобная запись без ожидания  
- 🧩 **Гибкость** — легко обновлять базу знаний при изменении прайса или появлении новых услуг  
- 🔒 **Надёжность** — Савви гарантирует 100% ответов и работу по сложным бизнес-сценариям
